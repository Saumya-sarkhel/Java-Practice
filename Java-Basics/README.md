# Java Programming Basics 🧑‍💻

This repository contains a collection of Java programs designed for beginners to get hands-on practice with core Java concepts. The examples are structured in a way to gradually build a foundational understanding of programming using Java.

## 📁 Project Structure

Each file in this repository demonstrates a key concept of Java programming:
`src` folder contains all the programs

| Folder Name                  | Description |
|---------------------------|-------------|
| Class 1                | A basic Java program to print hello world, Data types, and Addition of two numbers with and withoout user input. |

Make sure you have Java installed. You can compile and run any file using the following commands:

